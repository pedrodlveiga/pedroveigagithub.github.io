Aprendizado na Trybe

Este primeiro artigo é para descrever como está sendo o aprendizado nestas três últimas semanas no curso de Desenvolvimento WEB da Trybe.Na primeira semana aprendemos a como utilizar o terminal do Linux/MacOs, sistemas operacionais baseados no UNIX.Na semana seguinte, aprendemos a utilizar o gerenciador de versionamento Git e GitHub e iniciamos com o bloco seguinte sobre HTML e CSS.Nessa terceira semana, terminamos o bloco de HTML e CCS e estamos realizando nesse exato momento, o nosso primeiro projeto.

Na Trybe, além das habilidades técnicas, chamadas de hard skills, também aprendemos muito com as aulas de relacionamento interpessoal, mais conhecidas como soft skills. Os 02 tipos de habilidades, certamente, forma um profissional mais completo mais valorizado no mercado de trabalho.
